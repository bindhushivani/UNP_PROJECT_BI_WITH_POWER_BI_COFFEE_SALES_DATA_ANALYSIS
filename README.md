# UNP_PROJECT_BI_WITH_POWER_BI_COFFEE_SALES_DATA_ANALYSIS
I completed a Coffee Sales Data Analysis project in collaboration with UNP using real world coffee sales data 
The project involved cleaning, analysing and visualising customer and sales data using Microsoft Power Bi 
I created an interactive dashboard to show the jey insights such as non peak hours create more revenue than peak hours 
Also Afternoon & Night sales are slightly more than Morning sales 
I used KPI 's that describes the total revenue, peak hour & non peak hour revenues , morning - afternoon & night sales 
This project helped me to develop skills in data analysis, data visualization and dashboard creation using Power Bi
